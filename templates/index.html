Fundamentos da Programação em Python

Aprenda os conceitos da linguagem python

Introdução

Python é uma linguagem de programação, interpretada e com uma sintaxe simples e fácil de aprender. É amplamente utilizada em áreas como desenvolvimento web, ciência de dados e inteligência artificial.

Variáveis e Tipos de Dados

Variáveis armazenam valores que podem ser utilizados e modificados no programa. Os principais tipos de dados em Python são:

  • int: números inteiros (ex: 10)
  • float: números decimais (ex: 3.14)
  • str: textos (ex: "Olá Mundo")
  • bool: valores booleanos (True ou False)

Operadores

Python possui muitos Operadores:

  • Aritméticos: + - * / // % **
  • Comparação: == != > < >= <=
  • Lógicos: and, or, not

Estruturas Condicionais

Usadas para tomar decisões no código:

idade = 18
if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

Laços de Repetição

Usados para repetir instruções:

for i in range(5):
    print(i)

contador = 0
while contador < 5:
    print(contador)
    contador += 1

Funções

Funções são blocos de código reutilizáveis:

def saudacao(nome):
    print(f"Olá, {nome}!")

saudacao("Bruno")
